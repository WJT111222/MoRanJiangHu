> **分析日期**: 2026-07-09

---

## 功能概述

用户图库功能允许玩家将物品生图结果持久化存入 IndexedDB 独立 store（`user_image_gallery`），按「物品名称 + 题材模式 + 创意工坊模块ID」三元组唯一索引。后续生图时自动复用已有图库图片，避免重复调用 AI 生图接口。同时在图库管理弹窗（桌面端 + 移动端）中支持浏览、筛选、预览和删除。

## 移植策略建议

### 方案 A：精确 Cherry-Pick（推荐）

1. 先移植基础层：`utils/itemImage.ts` → `services/dbService.ts`（仅图库相关函数）

3. 最后移植 UI 层：InventoryModal → MobileInventoryModal → ImageManagerModal → MobileImageManagerModal → App.tsx

### 方案 B：整体移植提交 `00082fdb`

如果该提交未混入其他无关改动，可直接 `git cherry-pick 00082fdb`。

### 移植顺序

1. `utils/itemImage.ts`（无依赖）
2. `services/dbService.ts` 图库部分（依赖 types 和 models）
