# 多货币汇率系统（MCX）移植方案

> 从 Rebase 分支移植到 Currency 分支，共存模式，不影响 main 现有功能。

---

## 一、命名与定位

**多货币汇率系统**（Multi-Currency Exchange System，MCX）是 Rebase 分支相对于 main 新增的货币扩展层。它与 main 的"基础货币系统"通过配置中的 `systems` 字段是否存在来判断启用哪套。

```
profile.economy.currencySystem 结构：
{
  id: "xxx",                  // 基础货币系统（必填，兼容旧存档）
  units: [...],
  systems: { ... },           // MCX 多体系（可选）
  exchangeRatePolicies: { ... }  // MCX 汇率策略（可选）
}
```

let _expandedCurrencySystem: ExpandedCurrencySystem | undefined;
let _cachedProfileKey: string | undefined;

export function 是否多货币模式(profile: any): boolean {
    const systems = profile?.economy?.currencySystem?.systems;
    return !!(systems && Object.keys(systems).length > 0);
}

export function 获取展开货币系统(profile: any): ExpandedCurrencySystem | undefined {
    if (!是否多货币模式(profile)) {
        _expandedCurrencySystem = undefined;
        _cachedProfileKey = undefined;
        return undefined;
    }
    const systems = profile?.economy?.currencySystem?.systems;
    const policies = profile?.economy?.currencySystem?.exchangeRatePolicies;
    const profileKey = JSON.stringify({ systems, policies });
    if (_cachedProfileKey === profileKey && _expandedCurrencySystem) {
        return _expandedCurrencySystem;
    }
    _expandedCurrencySystem = 加载并展开货币系统({ systems }, policies);
    _cachedProfileKey = profileKey;
    return _expandedCurrencySystem;
}

export function 重置货币系统缓存(): void {
    _expandedCurrencySystem = undefined;
    _cachedProfileKey = undefined;
}
```

const 迁移货币桶旧单位名 = (货币桶: any, expandedSystem: ExpandedCurrencySystem | undefined): void => {
    if (!expandedSystem || !货币桶 || typeof 货币桶 !== 'object') return;
    for (const [systemId, units] of Object.entries(货币桶)) {
        if (!expandedSystem.systems[systemId]) continue;
        for (const unitId of Object.keys(units as any)) {
            if (expandedSystem.systems[systemId].units[unitId]) continue;
            // 通过 aliases 匹配新单位 ID
            const matched = Object.entries(expandedSystem.systems[systemId].units)
                .find(([_, u]) => u.aliases?.includes(unitId));
            if (matched) {
                console.warn(`[货币桶] 迁移: ${systemId}.${unitId} → ${matched[0]}`);
                (units as any)[matched[0]] = (units as any)[unitId];
                delete (units as any)[unitId];
            }
        }
    }
};
```

## 四、完整调用链

```
配置层        apiConfig.ts: 是否多货币模式 / 获取展开货币系统
    ↓
数据加载层    currencyTemplateExpander.ts / currencySystemLoader.ts
    ↓
运行时逻辑    economyRules.ts: 检查汇率 / 构建汇率生成提示
    ↓
提示词层      currencyReference.ts: 货币参考注入 / 比例参考 / 汇率快照
    ↓
系统集成      systemPromptBuilder.ts (快照分流 + 提示词注入)
              responseCommandProcessor.ts (地点变更 → 检查汇率)
              stateTransforms.ts (旧单位名迁移)
              variableModelWorkflow.ts (传 AI)
              variableModel.ts (消费汇率生成任务)
              auctionHouse.ts (null 返回)
    ↓
显示层        currencyDisplay.ts: 多系统货币快照
    ↓
数据模型      models/system.ts (8 个接口 + 可选字段)
              models/world.ts (汇率字段)
              models/character.ts (货币桶)
```
